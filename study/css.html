CSS Reference for Developers

Selectors

SelectorDescriptionNotes
*Universal selector; matches all elementsStandard
elementType selector; matches all elements of a type (p, div)Standard
.classClass selector; matches elements with the classStandard
#idID selector; matches a single element with the IDStandard
[attribute]Matches elements with the attributeStandard
[attribute=value]Attribute selector with specific valueStandard
[attribute~=value]Attribute contains wordStandard
[attribute|=value]Attribute starts with value (dash-separated)Standard
[attribute^=value]Attribute starts with valueStandard
[attribute$=value]Attribute ends with valueStandard
[attribute*=value]Attribute contains valueStandard
:hoverMatches element when hoveredStandard
:activeMatches element while active (pressed)Standard
:focusMatches element when focusedStandard
:first-childMatches first child element of a parentStandard
:last-childMatches last child element of a parentStandard
:nth-child(n)Matches nth child elementStandard
:nth-of-type(n)Matches nth child of typeStandard
::beforeInserts content before elementStandard
::afterInserts content after elementStandard
::first-lineTargets first line of textStandard
::first-letterTargets first letter of textStandard
:not(selector)Negation pseudo-classStandard
